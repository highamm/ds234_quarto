Exercise 2. Answers will vary.

When using Quarto, all code must run for the document to render. Therefore, someone else can (usually) run your code. Anyone can see exactly what was done in your analysis (if there were outliers removed, what data wrangling steps took place, etc.). This is not necessarily the case with R scripts and Word.
